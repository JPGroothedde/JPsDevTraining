Introduction

This API allows for interaction with the Account object contained in the application ORM. To call this API, the calling application needs to POST the following minimum parameters:

  • APIKEY -> The API key that allows for access to this API
  • OPERATION -> The operation that is to be performed

Implementation

Object Definition (Account)

Object Attributes
["FullName","FirstName","LastName","EmailAddress","Username","Password","ChangedBy"]
Object Relationships (Id for each associated object)
["UserRole_Id"]

Available Operations (CRUD)

The following base operations are available: (Additional Operations are defined at the bottom of this page)

  • READ
  • UPDATE
  • CREATE
  • DELETE
  • LIST

READ

Input Data (POST FIELDS):
{"APIKEY":"The Provided Api Key","OPERATION":"READ","AccountId":"[The Id To Retrieve]"}
Expected Result:
{"Result":"Success","Obj":"[json encoded Account Object]"}

UPDATE

Input Data (POST FIELDS):
{"APIKEY":"The Provided Api Key","OPERATION":"UPDATE","AccountId":"[The Id To Update]","FullName":"[new value]","FirstName":"[new value]","LastName":"[new value]","EmailAddress":"[new value]","Username":"[new value]","Password":"[new value]","ChangedBy":"[new value]","UserRole_Id":"[Id of UserRole Object to associate]"}
Expected Result:
{"Result":"Success","Message":"Object Modified"}

CREATE

Input Data (POST FIELDS):
{"APIKEY":"The Provided Api Key","OPERATION":"CREATE","FullName":"[new value]","FirstName":"[new value]","LastName":"[new value]","EmailAddress":"[new value]","Username":"[new value]","Password":"[new value]","ChangedBy":"[new value]","UserRole_Id":"[Id of UserRole Object to associate]"}
Expected Result:
{"Result":"Success","ObjId":"[Id of newly created object]"}

DELETE

Input Data (POST FIELDS):
{"APIKEY":"The Provided Api Key","OPERATION":"DELETE","AccountId":"[The Id To Delete]"}
Expected Result:
{"Result":"Success"}

LIST

Input Data (POST FIELDS):
{"APIKEY":"The Provided Api Key","OPERATION":"LIST","QueryConditions":"[The Query Conditions]"}
Expected Result:
{"Result":"Success","ObjArray":"[Array Of JSON Encoded Objects]"}

Example Query Conditions

array("AND" => ["ObjectAttribute","<",[Value]],"AND" => ["ObjectAttribute","is not null"],"OR" => ["ObjectAttribute","=",[Value]],"AND" => ["ObjectAttribute","is null"])

All available compare symbols
"=","!=",">","<",">=","<=","in","not in","like","not like","is null","is not null","between","not between"
